# Hábitos Serx

Aplicación personal de hábitos saludables orientada a alimentación, ejercicio y rutinas diarias.

## Objetivo del proyecto

El objetivo de esta app es acompañar el seguimiento de hábitos personales de salud de una forma sencilla, clara y motivadora.

El proyecto forma parte de una línea de aplicaciones personalizadas pensadas para convertir rutinas reales en herramientas prácticas, visuales y adaptadas a la vida cotidiana del usuario.

## Características principales

- Seguimiento de hábitos saludables.
- Organización de rutinas de alimentación y ejercicio.
- Enfoque personal y adaptado al usuario.
- Interfaz ligera para uso diario.
- Proyecto orientado a navegador.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Estado del proyecto

Proyecto funcional en fase inicial de aprendizaje y mejora continua.

## Aprendizajes trabajados

Con este proyecto se practican conceptos como:

- Diseño de una app personal de hábitos.
- Organización de información relacionada con salud y rutinas.
- Interacción básica con JavaScript.
- Creación de interfaces sencillas y útiles.
- Desarrollo de herramientas personalizadas basadas en necesidades reales.
